
# Modular Content Block Specifications

This document provides an overview of all modular content blocks included in the website project, including their purpose and fields.

---

## Common & Shared Elements

### Headers
- **Eyebrow**: Small text displayed above the main header
- **Header**: The main header
- **Subheader**: Supporting text shown below the header

### Call-to-action Buttons (CTAs)
- **Text**
- **Link**: internal or external
- **Style Variant**: e.g., primary/secondary
- **Button Type**: solid, hollow, transparent
- **Size**: large or small

### Component Settings
- **Spacing**
- **Background Color**
- **Text Color**

---

## Content Blocks

### Hero Block
A prominent banner with visuals and compelling text.
**Best used for**: page introductions, campaign launches, key announcements, product showcases.

**Fields**:
- Headers
- Main body text
- Background (image or video)
- Background overlay opacity
- Size (full or half screen)
- Call-to-action buttons

---

### Text Block
Single or two-column layout for structured content.
**Best used for**: general content, articles, services, about.

**Fields**:
- Headers
- Main body text (WYSIWYG)
- Call-to-action buttons
- Component settings

---

### Image Block
Showcase key visuals with adjustable width.
**Best used for**: product photos, team portraits, illustrations.

**Fields**:
- Image file
- Image width (wide or narrow)
- Component settings

---

### Video Block
Embed hosted or local videos.
**Best used for**: tutorials, promos, product demos.

**Fields**:
- Video source (embed/upload)
- Video width
- Component settings

---

### Accordion Block
Expandable sections for long-form content.
**Best used for**: FAQs, specifications, service/process details.

**Fields**:
- Headers
- Accordion items (header, WYSIWYG text, CTAs)
- Behavior (single/multiple open)
- Component settings

---

### Quote Block
Stylish block for quotes or testimonials.
**Best used for**: testimonials, insights, team highlights.

**Fields**:
- Quote
- Quotee
- Component settings

---

### Gallery Block
Multi-image grid display.
**Best used for**: portfolios, product galleries, event recaps.

**Fields**:
- Headers
- Images
- Component settings

---

### Form Block
Embed contact or registration forms.
**Best used for**: contact, feedback, lead collection.

**Fields**:
- Headers
- Form selection
- Component settings

---

### Side by Side Block
Two-column layout mixing various content types.
**Best used for**: comparisons, dual presentations.

**Fields**:
- Headers
- Content type for left/right (Text, Image, Video, Quote, Accordion, Form)
- Column ratio (50/50, 70/30, 30/70)
- Direction (LTR or RTL)
- Component settings

---

### Archive Block
Dynamic listing of posts or pages.
**Best used for**: blogs, directories, resource catalogs.

**Fields**:
- Headers
- Archive type (pages/posts)
- Items per page
- Pagination toggle
- Component settings

---

### Text & Image Block
Text with image, side by side.
**Best used for**: storytelling, product highlights, team profiles.

**Fields**:
- Headers
- Body text (WYSIWYG)
- Image
- Image position (left/right)
- Optional CTAs
- Component settings

---

### Text & Video Block
Text and embedded video side by side.
**Best used for**: demonstrations, tutorials, promotional content.

**Fields**:
- Headers
- Body text (WYSIWYG)
- Video (embed/upload)
- Video position (left/right)
- Optional CTAs
- Component settings

---

### Slider Block
Horizontal slider with repeatable items.
**Best used for**: features, testimonials, references.

**Fields**:
- Headers
- Slides (image, title, optional link)
- Component settings
